Important Notes:

  • href="..." - Specifies the URL of the page the link goes to
  • target="_blank" - Opens link in a new tab/window
  • target="_self" - Opens link in the same frame (default)
  • target="_top" - Opens link in the full window
  • target="_parent" - Opens link in the parent frame

  • If the links won't work, it's your browser issue.
